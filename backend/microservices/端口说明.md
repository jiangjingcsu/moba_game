# 微服务端口说明

## 端口总览

| 端口号 | 服务 | 用途 |
|--------|------|------|
| 3306 | moba-business | MySQL 数据库连接 |
| 6379 | 多服务共用 | Redis 连接（通过不同 database 编号隔离） |
| 8080 | moba-gateway | Spring Cloud Gateway HTTP 入口 |
| 8081 | moba-business | Spring Boot HTTP 端口 |
| 8084 | moba-data | Spring Boot HTTP 端口 |
| 8888 | moba-battle | Netty WebSocket 端口（客户端连接，路径 `/ws/battle`） |
| 8889 | moba-battle | Netty TCP 端口（战斗内部通信） |
| 9998 | moba-match | Netty WebSocket 端口（客户端连接，路径 `/ws/match`） |
| 9999 | moba-gateway-netty | Netty WebSocket 端口（游戏网关入口，路径 `/ws/game`） |
| 20782 | moba-match | Dubbo RPC 端口 |
| 20881 | moba-business | Dubbo RPC 端口 |
| 20883 | moba-battle | Dubbo RPC 端口 |
| 20884 | moba-data | Dubbo RPC 端口 |
| 27017 | moba-data | MongoDB 连接 |

---

## 各服务端口详情

### moba-match（匹配服务）

| 端口号 | 用途 | 配置键 | 说明 |
|--------|------|--------|------|
| 9998 | Netty WebSocket | `match.websocket.port` | 客户端通过 WebSocket 连接匹配服务，路径 `/ws/match` |
| 20782 | Dubbo RPC | `dubbo.protocol.port` | 提供 Dubbo RPC 服务，供其他服务调用匹配接口 |
| 6379 | Redis | `redis.port` | Redis 连接，database=1 |

### moba-battle（战斗服务）

| 端口号 | 用途 | 配置键 | 说明 |
|--------|------|--------|------|
| 8888 | Netty WebSocket | `battle.port` | 客户端通过 WebSocket 连接战斗服务，路径 `/ws/battle` |
| 8889 | Netty TCP | `battle.battle-server-port` | 战斗内部 TCP 通信端口，用于战斗服务器间数据交换 |
| 20883 | Dubbo RPC | `dubbo.protocol.port` | 提供 Dubbo RPC 服务，供其他服务调用战斗接口 |
| 6379 | Redis | `spring.data.redis.port` | Redis 连接，database=2 |

### moba-gateway（Spring Cloud Gateway 网关）

| 端口号 | 用途 | 配置键 | 说明 |
|--------|------|--------|------|
| 8080 | HTTP/WebSocket | `server.port` | Spring Cloud Gateway 入口端口，处理 HTTP 请求和 WebSocket 路由 |
| 6379 | Redis | `spring.data.redis.port` | Redis 连接，database=0 |

### moba-gateway-netty（Netty 网关）

| 端口号 | 用途 | 配置键 | 说明 |
|--------|------|--------|------|
| 9999 | Netty WebSocket | `server.port` | 游戏网关主入口，路径 `/ws/game`，内部路由到匹配或战斗服务 |

### moba-business（业务服务）

| 端口号 | 用途 | 配置键 | 说明 |
|--------|------|--------|------|
| 8081 | HTTP | `server.port` | Spring Boot HTTP 端口，提供 REST API |
| 20881 | Dubbo RPC | `dubbo.protocol.port` | 提供 Dubbo RPC 服务，供其他服务调用业务接口 |
| 3306 | MySQL | `spring.datasource.url` 内嵌 | MySQL 数据库连接 |
| 6379 | Redis | `spring.data.redis.port` | Redis 连接，database=0 |

### moba-data（数据服务）

| 端口号 | 用途 | 配置键 | 说明 |
|--------|------|--------|------|
| 8084 | HTTP | `server.port` | Spring Boot HTTP 端口，提供 REST API |
| 20884 | Dubbo RPC | `dubbo.protocol.port` | 提供 Dubbo RPC 服务，供其他服务调用数据接口 |
| 27017 | MongoDB | `spring.data.mongodb.port` | MongoDB 数据库连接 |
| 6379 | Redis | `spring.data.redis.port` | Redis 连接，database=3 |

---

## Redis 数据库隔离

所有服务共用同一个 Redis 实例，通过不同 database 编号隔离：

| 服务 | Redis Database |
|------|---------------|
| moba-gateway | 0 |
| moba-business | 0 |
| moba-match | 1 |
| moba-battle | 2 |
| moba-data | 3 |

---

## 端口分配规律

- **HTTP 端口**：8080 ~ 8084，按服务递增
- **Netty WebSocket 端口**：8888、8889（battle），9998（match），9999（gateway-netty）
- **Dubbo RPC 端口**：20782（match），20881（business），20883（battle），20884（data）
- **外部服务端口**：3306（MySQL），6379（Redis），27017（MongoDB）
